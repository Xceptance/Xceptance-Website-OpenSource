Your Responsibilities and Tasks

  • Create, maintain and execute manual test cases within software development projects
  • Report, evaluate, and manage bug entries and tickets
  • Design test concepts
  • Develop and execute automated tests
  • Execute performance and stress tests

Minimum Qualifications

  • Degree in computer science, business informatics, or the like
  • Knowledge of test methods, tools, and processes
  • Knowledge of basic Web technologies such as HTTP/S, HTML, CSS and JavaScript
  • Experience in using issue trackers
  • Familiarity with test automation tools for web applications such as Selenium, WebDriver or WATIR
  • Good English and German language skills

Desired Skills

  • Positive attitude with a drive to learn
  • Commitment and reliability
  • Ability to work in a team and a dynamic environment
  • Excellent and persuasive communication skills
  • Analytical skills and problem-solving abilities
  • Attention to detail and appreciation of quality assurance
